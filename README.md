# Garden Tools DoubleDown

Garden Tools DoubleDown은 선택한 폴더 안에서 중복 파일을 찾아 검토하고 삭제할 수 있는 Windows용 데스크톱 유틸리티입니다. 파일 크기, 일부 해시, 전체 SHA-256 해시 순서로 후보를 좁혀 실제 내용이 같은 파일만 중복으로 묶습니다.

## 주요 기능

- 여러 폴더를 재귀적으로 스캔
- 최소/최대 파일 크기 필터
- 포함할 확장자와 제외할 확장자 지정
- 숨김 파일 포함 여부 선택
- 크기 기반 사전 필터, 64KB 부분 해시, 전체 SHA-256 해시를 이용한 중복 탐지
- 해시 캐시를 사용해 다음 스캔 속도 개선
- 중복 그룹별 보관 파일 자동 추천
- 선택한 파일을 휴지통으로 이동하거나 영구 삭제
- 실제 삭제 없이 결과만 확인하는 드라이런 모드
- 삭제 로그 저장
- 중복 결과 CSV 내보내기

## 바로 실행하기

빌드된 실행 파일이 포함되어 있다면 아래 파일을 실행합니다.

```powershell
& ".\dist\Garden Tools DoubleDown.exe"
```

설정, 해시 캐시, 삭제 로그는 사용자 홈 폴더 아래에 저장됩니다.

```text
%USERPROFILE%\.garden_tools_doubledown\
```

## 개발 환경에서 실행하기

### 요구 사항

- Python 3.10 이상 권장
- Windows

### 설치

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### 실행

```powershell
python main.py
```

## 사용 방법

1. 왼쪽의 검색 폴더 영역에서 스캔할 폴더를 추가합니다.
2. 필요한 경우 스캔 필터를 설정합니다.
   - 최소 크기 / 최대 크기
   - 포함 확장자: 예시 `.jpg .png .mp4`
   - 제외 확장자: 예시 `.tmp .log`
   - 숨김 파일 포함
3. `스캔 시작`을 눌러 중복 파일을 찾습니다.
4. 오른쪽 결과 목록에서 삭제할 파일을 검토하고 선택합니다.
5. 삭제 옵션을 확인한 뒤 `선택 파일 삭제`를 실행합니다.

기본 삭제 방식은 휴지통 이동입니다. 영구 삭제 옵션을 사용할 때는 복구가 어려울 수 있으니 드라이런으로 먼저 확인하는 것을 권장합니다.

## 중복 판별 방식

Garden Tools DoubleDown은 모든 파일을 처음부터 끝까지 해시하지 않고, 단계적으로 후보를 줄입니다.

1. 파일 크기가 같은 파일만 후보로 유지합니다.
2. 후보 파일의 앞 64KB를 MD5로 해시해 다시 그룹화합니다.
3. 남은 후보의 전체 내용을 SHA-256으로 해시합니다.
4. 전체 SHA-256 값이 같은 파일만 중복 그룹으로 표시합니다.

해시 결과는 파일 경로와 수정 시간을 기준으로 SQLite 캐시에 저장됩니다. 변경되지 않은 파일은 다음 스캔에서 다시 읽지 않아도 됩니다.

## 보관 파일 선택 기준

중복 그룹 안에서 기본으로 남길 파일은 다음 기준으로 정렬됩니다.

1. 선호 폴더에 있는 파일
2. 생성 시간이 더 오래된 파일
3. 파일명이 더 짧은 파일

현재 UI에서는 선호 폴더 설정 항목이 별도로 노출되어 있지 않지만, 설정 파일에는 `prefer_folders` 값이 준비되어 있습니다.

## CSV 내보내기

스캔 결과가 있을 때 `CSV 내보내기`를 사용하면 중복 그룹, 보관/삭제 상태, 파일명, 크기, 경로, SHA-256 값을 CSV 파일로 저장할 수 있습니다.

## 삭제 로그

드라이런이 아닌 삭제 작업을 실행하면 삭제 대상, 크기, 해시, 처리 방식, 처리 시각이 JSON 로그로 저장됩니다.

```text
%USERPROFILE%\.garden_tools_doubledown\delete_log.json
```

## 빌드

PyInstaller로 단일 실행 파일을 만들 수 있습니다.

```powershell
pip install pyinstaller
pyinstaller GardenToolsDoubleDown.spec
```

빌드가 끝나면 실행 파일은 아래 경로에 생성됩니다.

```text
dist\Garden Tools DoubleDown.exe
```

## 프로젝트 구조

```text
.
├── main.py                  # 애플리케이션 진입점
├── requirements.txt         # 런타임 의존성
├── GardenToolsDoubleDown.spec # PyInstaller 빌드 설정
└── src
    ├── core                 # 스캔, 해시, 중복 탐지, 삭제 작업
    ├── models               # 파일/중복 그룹 데이터 모델
    ├── ui                   # PyQt6 UI 패널과 메인 윈도우
    └── utils                # 설정 저장소와 해시 캐시
```

## 의존성

- PyQt6
- send2trash

## 주의 사항

- 시스템 폴더, `.git`, `node_modules`, `__pycache__` 등 일부 디렉터리는 기본적으로 스캔에서 제외됩니다.
- 0바이트 파일은 스캔 대상에서 제외됩니다.
- 휴지통 이동이 기본값이지만, 드라이브나 환경에 따라 휴지통 처리가 실패할 수 있습니다.
- 영구 삭제는 되돌리기 어렵습니다. 중요한 폴더를 정리할 때는 드라이런과 CSV 내보내기로 먼저 확인하세요.
